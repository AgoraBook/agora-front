## :white_check_mark: Requirements

프로젝트를 시작하기 전에 Git, NodeJS, Npm을 설치해 주세요 ⭐

## :checkered_flag: Starting

```bash
# 프로젝트를 복제하세요
$ git clone https://github.com/AgoraBook/agora-front

# 폴더 이동
$ cd agora-front

# 디펜던시 설치
$ npm install

# 프로젝트 실행
$ npm run dev

# 이 서버는 다음의 포트에서 실행됩니다. <http://localhost:5174>
```
